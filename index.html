Java Solutions to Problems from Cracking the Coding Interview, Fourth Edition

Problems and solutions from Cracking the Coding Interview, Fourth Edition: 150 Programming Interview Questions and Solutions by Gayle Laakmann McDowell. The questions are solved in Java 7, with JUnit tests. This a Maven project, and can be opened in Eclipse, with the m2e plugin installed.

Includes a Java class called CStyleString, which allows some of the C string algorithms to be solved in Java.
